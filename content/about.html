About the site

The most recent design of the site spun out of my desire to create a website without using Bootstrap for the first time in 6 years - forcing me to learn this flexbox thing that had appeared in the meantime.

I plan to write an article about the configuration and optimisation of the site, but for now here are the key parts:

  • Generated using Nanoc. Even though I have to use extensive Stack Overflow Driven Development to wrangle the Ruby, it's worth it for this well designed static site generator.

  • Hosted on Amazon S3, fronted by CloudFront with an ACM certificate.
  • Uploaded using s3sup, S3 static website uploader. A side project of mine.
